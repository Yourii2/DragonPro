# Release Notes 1.2.7

- إضافة إعداد جديد للتحكم في سعر الشراء المستخدم داخل إذن الاستلام وإذن المرتجع.
- إضافة صفحة `تأكيد الاوردرات` ضمن القائمة الرئيسية مع دعم التشغيل السريع بالباركود.
- دعم إسناد أوردرات `قيد الانتظار` و`مرتجع` فقط إلى المندوب داخل مسار التأكيد.
- دعم تجهيز القرارات محلياً إلى `أوردرات مؤكدة` و`أوردرات مؤجلة` و`أوردرات ملغية` قبل تنفيذ الحفظ النهائي.
- إضافة أدوات سريعة للحفظ والمسح والطباعة والتحديث داخل صفحة تأكيد الأوردرات.
- تحسين رسائل التعارض والحظر عند وجود الأوردر مع مندوب آخر أو داخل اليومية المفتوحة.
- تحسينات بصرية على بطاقات الملخص والتنفيذ لتناسب شاشة التشغيل اليومية.

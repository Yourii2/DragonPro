# Release Notes 1.3.1

- تحسين واجهة إذن الاستلام بإظهار حقل واضح بعنوان `سعر التكلفة الجديد` لكل صنف.
- إضافة عرض `متوسط التكلفة المتوقع` لحظيًا قبل الحفظ بناءً على الكمية الحالية + الكمية المستلمة.
- اعتماد تحديث `cost_price` بمتوسط مرجّح (Weighted Average Cost) في backend عند حفظ إذن الاستلام.
- إضافة أدوات مباشرة في صفحة `تأكيد الاوردرات`:
  - حذف الأوردر من قائمة `الأوردرات الحالية` للمندوب.
  - استرجاع الأوردر من قائمة `الأوردرات الملغية` مع إعادة حالته السابقة.
- ضبط دورة اليومية:
  - منع الإغلاق إذا لا توجد يومية مفتوحة.
  - السماح بالإغلاق بمبلغ `0` بدون تسوية خزينة.
  - إغلاق جميع اليوميات المفتوحة للمندوب عند التقفيل النهائي لتفادي بقاء يوميات قديمة.
